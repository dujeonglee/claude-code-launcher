# Requirements

Core Dependencies:
```
PySide6>=6.5.0
requests>=2.31.0
PyYAML>=6.0.1
```

Development Dependencies (optional):
```
mypy>=1.0.0
pylint>=3.0.0
pytest>=7.0.0
```

## Installation

```bash
pip install -r requirements.txt
```

## Runtime Requirements

### System
- Python 3.9 or higher
- Qt6 libraries (included with PySide6 wheel)

### For Claude Code CLI
- Node.js 18+
- npm (comes with Node.js)

### For LLM Servers
- Ollama, vLLM, MLX Serve, llama.cpp, LM Studio, or TGI
- Each with appropriate models loaded
